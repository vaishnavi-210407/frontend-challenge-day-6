🚀 Day 6 / 21 — To-Do App (Basic)
📌 Project Name

To-Do App (Basic)

🎯 Goal

Build a simple and interactive To-Do app where users can add and delete tasks dynamically.

🧩 Features
➕ Add new tasks
❌ Delete tasks
📋 Dynamic task list rendering
⚡ Instant UI updates
📱 Responsive design
🔄 Step-by-Step Flow
User enters a task in the input field
User clicks the “Add” button
JavaScript reads the input value
Task is added to an array
Task list is rendered on the UI
User clicks delete button
Selected task is removed from the list
🖥 UI Plan
Input field
Add button
Task list container
Delete button for each task
⚙️ Logic Plan
Add event listener on “Add” button
Store tasks in an array
Loop through array to display tasks
Add delete functionality using index or ID
Update UI after every change
🚧 Edge Cases
Empty input submission
Duplicate tasks
Long text handling
Fast multiple clicks
🛠 Tech Stack
HTML5
CSS3
JavaScript (Vanilla JS)
